# Expense Tracker Dashboard

A finance dashboard built using PHP, MySQL, and Chart.js to track expenses and visualize spending.

## Features
- Add expenses
- Budget tracking
- Category spending charts
- Dark gold finance dashboard UI
- Interactive charts using Chart.js

## Tech Stack
- PHP
- MySQL
- Chart.js
- HTML
- CSS
- JavaScript

## How to Run

1. Install XAMPP or any Apache + MySQL server
2. Move the project folder to `htdocs`
3. Start Apache and MySQL
4. Open in browser:

http://localhost/expense-tracker/dashboard.php
